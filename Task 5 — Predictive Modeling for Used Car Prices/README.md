# 🚀 Task 5 — Predictive Modeling for Used Car Prices

## 🎯 Goal

Build a Machine Learning regression model to predict used car prices based on vehicle features.

---

# ⚙️ Model Pipeline

## 1. Data Preparation
- Selected important features
- Encoded categorical variables
- Split dataset into training and testing sets

## 2. Model Building
- Applied Linear Regression
- Applied Random Forest Regressor
- Trained models on dataset

## 3. Model Evaluation
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

---

# 📊 Key Findings

- Mileage and manufacturing year strongly impact price
- Random Forest performed better than Linear Regression
- Machine learning effectively predicts used car prices

---

# 🛠️ Libraries Used

- Scikit-learn
- Pandas
- NumPy

Successfully developed a predictive model for used car price estimation.
