# 🚗 Used Car Price Analysis & Prediction

## 📌 Project Overview

This project demonstrates a complete Data Science and Machine Learning workflow using a Used Car Dataset. The project focuses on analyzing vehicle data, identifying important trends, visualizing patterns, and predicting car prices using machine learning techniques.

The objective is to understand how factors such as mileage, fuel type, transmission, year, and condition affect used car prices.
